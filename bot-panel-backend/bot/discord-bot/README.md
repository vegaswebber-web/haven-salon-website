# Discord Bot

This project is a Discord bot built with Node.js and TypeScript. It provides various commands for managing server interactions, including music playback, user management, and channel control.

## Features

- **Music Command**: Play songs from YouTube or Spotify using the `.music` command.
- **Avatar Command**: Retrieve and display a user's profile picture with the `.avatar` command.
- **Ban Command**: Ban users from the server using the `.ban` command.
- **Delete Command**: Delete a specified number of messages from a channel with the `.delete` command.
- **Unban Command**: Unban users from the server using the `.unban` command.
- **Lock Command**: Lock a channel to prevent further messages with the `.lock` command.
- **Help Command**: Display a list of available commands and their usage with the `.help` command.
- **Mute Command**: Temporarily mute users to prevent them from sending messages with the `.mute` command.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd discord-bot
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Create a `.env` file in the root directory and add your Discord bot token:
   ```
   DISCORD_TOKEN=your_bot_token_here
   ```

## Usage

To start the bot, run the following command:
```
npm start
```
